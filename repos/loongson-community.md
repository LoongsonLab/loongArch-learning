# 开源平台

## OpenHarmony-LoongArch

该仓库介绍龙开源鸿蒙(OHOS)龙架构(LoongArch)主线移植项目。

网址链接：https://gitee.com/openharmony-loongarch

## Loongson

龙芯官方帐号，支持并维护多个开源项目，包括有：
- Firmware，维护、更新和发布固件。
- GCC，提供编译器维护与更新。
- QEMU，提供 LoongArch 架构支持。

以及其他开源软件和工具。

Github 主页链接：https://github.com/loongson

Gitee 主页链接：https://gitee.com/loongson

## Loongson Community

龙芯开源社区在 Github 中的帐号，支持并维护多个开源项目，包括有：
- Loongoffice，更新和维护开源办公处理软件。
- open-la500，更新和维护 LoongArch 架构下的开源处理器。
- U-boot，维护开发固件。

以及其他开源软件和工具。

帐号主页链接：https://github.com/loongson-community

## Loongson-Edu

龙芯教育开源仓库，主要提供基于 la32r 指令的教育生态技术支持，包括有：
- la32r-toolchains ，面向 la32r 指令集的交叉编译器。
- la32r-linux，基于 la32r 的 linux 内核。
- la32r-uboot，基于 la32r 的系统固件。

以及其他面向 la32r 的软件和工具。

帐号主页链接：https://gitee.com/loongson-edu

## Loongson-Cloud-Community

龙芯云社区，主要支持并维护 LoongArch 架构下云相关的软件与服务，包括有：
- k3s
- docker-library
- kafka

以及其他面向云生态的应用软件

帐号主页链接：https://github.com/Loongson-Cloud-Community
