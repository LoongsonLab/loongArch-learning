# 问题反馈

在使用本网站的课程视频、教材、实验和代码学习或授课的过程中，如果你遇到下面这些情况，欢迎到社区提问或反馈：

- 学习过程中有不理解的地方，希望得到解释或补充说明；
- 发现内容错误或表述不清楚（教材、PPT、实验指导书、代码等）；
- 代码 / 实验环境无法正常运行，或者运行结果与预期不一致；
- B 站、GitHub 等链接失效；
- 希望补充新的资源或改进现有资源的呈现方式。
