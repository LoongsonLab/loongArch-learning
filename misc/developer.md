# 龙芯爱好者社区

龙芯爱好者社区，旨在为想要在龙架构生态中进行软硬件开发的工程师们提供帮助与支持，包含有：
- 操作系统
- 龙架构教学与竞赛
- 硬件与固件
- 开源项目交流

以及其他方面的讨论交流。

网址链接：https://www.loongbbs.cn/

# loongson生态创新中心

龙芯生态创新衷心，包含有：
- helm LoongArch 源
- MongoDB 支持 LoongArch 架构版
- openstack 支持 LoongArch 架构版

以及其他前沿创新应用。

网址链接：https://gitee.com/cn-loongson
